Pro Max Calculator Web App
An advanced and interactive Calculator Web Application built using HTML, CSS, and JavaScript.
This project goes beyond a basic calculator by including scientific functions, history tracking, theme switching, and keyboard support.
Features
Basic Operations:
 Addition (+)
 Subtraction (−)
 Multiplication (×)
 Division (÷)
Scientific Functions:
 Square (x²)
 Square Root (√)
 Percentage (%)
 Brackets ( )
History Panel:
 Stores previous calculations
 Displays expressions with results
Theme Toggle:
 Dark Mode
 Light Mode
Keyboard Support:
 Numbers & operators input
 Enter → Calculate
 Backspace → Delete
 Escape → Clear
Technologies Used:
 HTML5
 CSS3 (Grid + Flexbox + Animations)
 JavaScript (DOM Manipulation)
